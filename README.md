# Smart Multi-Hazard Detection and Real-Time Alert System

## 📌 Problem
Many accidents happen due to fire, gas leaks, and high temperature.

## 💡 Solution
Our system detects multiple hazards in real-time and alerts users instantly.

## ⚙️ Features
- Fire Detection
- Gas Leak Detection
- Temperature Monitoring
- Real-time Alerts

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Data: Simulated Sensor Data
